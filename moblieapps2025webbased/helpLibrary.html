Help

Add Item: Use add item to add items to the website that the library owns. In order to check out items, you must add them first.
Example usage: The library ordered new books and they got delivered.

Remove Item: Use remove item to remove items that the library does not use anymore.
Example usage: The library lost a book.

Check an Item Out: Use this to check in stock items out.
Example usage: A person wants to check a book out.

Change Status to Returned: Use this to return items back in stock.
Example usage: A person wants to return a book.
